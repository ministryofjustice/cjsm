Administering CJSM

Features

Key features Other features

We've added a number of new features which should make it easier for organisation administrators to perform their functions and help to keep the CJSM service safe, secure and efficient.

Key features

Our new key features include:

  • Single sign on
  • Interactive training
  • Assign organisation administrator
  • Create sub organisations

Other features

Other features to note include:

  • Multiple details change in CJSM directory
  • Bulk add users
  • User password resets
  • Mark users as 'ex directory'
  • Self-suspension
  • Create a distribution list
